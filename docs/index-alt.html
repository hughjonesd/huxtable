Huxtable

Huxtable is a package for creating HTML and LaTeX tables. It provides similar functionality to xtable, but does a lot more, with a simpler interface. Here's a quick example. 

library(huxtable)
ht <- hux(Employee = c('John Smith', 'Jane Doe', 'David Hugh-Jones'), Salary = c(50000, 50000, 40000),
      add_colnames = TRUE)
bold(ht)[1,]           <- TRUE
bottom_border(ht)[1,]  <- TRUE
align(ht)[,2]        <- 'right'
right_padding(ht)      <- 10
left_padding(ht)       <- 10

ht
Employee Salary
John Smith 50000.00
Jane Doe 50000.00
David Hugh-Jones 40000.00
