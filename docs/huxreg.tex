\subsection{\texorpdfstring{Regression tables with
\texttt{huxreg}}{Regression tables with huxreg}}\label{regression-tables-with-huxreg}}

Huxtable includes the function \texttt{huxreg} to build a table of
regressions.

You call \texttt{huxreg} with a list of models. These models can be of
any class which has a \texttt{tidy} method defined in the
\href{https://cran.r-project.org/?package=broom}{broom} package. The
method should return a list of regression coefficients with names
\texttt{term}, \texttt{estimate}, \texttt{std.error} and
\texttt{p.value}. That covers most standard regression packages.

Let's start by running some regressions to predict a diamond's price.

\begin{Shaded}
\begin{Highlighting}[]
\KeywordTok{data}\NormalTok{(diamonds, }\DataTypeTok{package =} \StringTok{"ggplot2"}\NormalTok{)}
\NormalTok{diamonds <-}\StringTok{ }\NormalTok{diamonds[}\DecValTok{1}\OperatorTok{:}\DecValTok{100}\NormalTok{,]}

\NormalTok{lm1 <-}\StringTok{ }\KeywordTok{lm}\NormalTok{(price }\OperatorTok{~}\StringTok{ }\NormalTok{carat }\OperatorTok{+}\StringTok{ }\NormalTok{depth, diamonds)}
\NormalTok{lm2 <-}\StringTok{ }\KeywordTok{lm}\NormalTok{(price }\OperatorTok{~}\StringTok{ }\NormalTok{depth }\OperatorTok{+}\StringTok{ }\KeywordTok{factor}\NormalTok{(color, }\DataTypeTok{ordered =} \OtherTok{FALSE}\NormalTok{), diamonds)}
\NormalTok{lm3 <-}\StringTok{ }\KeywordTok{lm}\NormalTok{(}\KeywordTok{log}\NormalTok{(price) }\OperatorTok{~}\StringTok{ }\NormalTok{carat }\OperatorTok{+}\StringTok{ }\NormalTok{depth, diamonds)}
\end{Highlighting}
\end{Shaded}

You can override any statistics returned by \texttt{tidy} or
\texttt{glance}.

If you want to completely replace the output of tidy, use the
\texttt{tidy\_replace()} function. For example, here's how to print
different coefficients for a multinomial model.

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{mnl <-}\StringTok{ }\NormalTok{nnet}\OperatorTok{::}\KeywordTok{multinom}\NormalTok{(gear }\OperatorTok{~}\StringTok{ }\NormalTok{mpg, mtcars)}
\end{Highlighting}
\end{Shaded}

\begin{verbatim}
## # weights:  9 (4 variable)
## initial  value 35.155593 
## iter  10 value 23.131901
## final  value 23.129234 
## converged
\end{verbatim}

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{tidied <-}\StringTok{ }\NormalTok{broom}\OperatorTok{::}\KeywordTok{tidy}\NormalTok{(mnl)}
\NormalTok{models <-}\StringTok{ }\KeywordTok{list}\NormalTok{()}
\NormalTok{models[[}\StringTok{"4 gears"}\NormalTok{]] <-}\StringTok{ }\KeywordTok{tidy_replace}\NormalTok{(mnl, tidied[tidied}\OperatorTok{$}\NormalTok{y.level }\OperatorTok{==}\StringTok{ }\DecValTok{4}\NormalTok{, ])}
\NormalTok{models[[}\StringTok{"5 gears"}\NormalTok{]] <-}\StringTok{ }\KeywordTok{tidy_replace}\NormalTok{(mnl, tidied[tidied}\OperatorTok{$}\NormalTok{y.level }\OperatorTok{==}\StringTok{ }\DecValTok{5}\NormalTok{, ])}
\KeywordTok{huxreg}\NormalTok{(models, }\DataTypeTok{statistics =} \StringTok{"AIC"}\NormalTok{)}
\end{Highlighting}
\end{Shaded}
